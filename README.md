# ♟️ Pygame Chess

A fully playable chess game developed using Python and Pygame.

## Features

- Complete chessboard implementation
- Legal move validation
- Check detection
- Checkmate detection
- Stalemate detection
- Pawn promotion
- Turn-based gameplay
- Visual move highlighting
- Piece selection system

## Technologies Used

- Python
- Pygame

## Installation

Clone the repository:

```bash
git clone https://github.com/nexus1201/pygame-chess.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the game:

```bash
python main.py
```
